The ways to open developers console in Mozilla Firefox and Safari.

Mozilla Firefox

  • Press Ctrl + Shift + I on Windows/Linux or Cmd + Option + I on Mac.
  • Right-click on the webpage and select "Inspect" from the context menu.
  • Go to the menu in the top right corner of the browser, select "Web Developer", and then click on "Web Console".

Safari

  • Press Ctrl + Shift + I on Windows/Linux or Cmd + Option + I on Mac.
  • Right-click on the webpage and select "Inspect" from the context menu.
  • Go to the menu in the top right corner of the browser, select "Web Developer", and then click on "Web Console".

Absolute and Relative links

Absolute links use cases

  • When linking to a page on a different domain.
  • When linking to a specific page that may be moved or renamed frequently.
  • When sharing links externally or across different platforms.
  • When linking to resources like images, stylesheets, or scripts hosted on a different domain.

Relative links use cases

  • When linking within the same website or domain.
  • When linking to pages within the same directory or subdirectory.
  • When creating a portable website that can be easily moved to a different server without breaking links.

Image formats

JPG(JPEG)

  • JPG is image format for photographs and complex images.
  • It uses lossy compression, which reduces file size by sacrificing some image quality.
  • It does not support transparency.
  • JPG supports millions of colors.

GIF

  • GIF is commonly used for simple images, animations, and graphics with flat colors.
  • It supports transparency and animation.
  • GIF images are limited to 256 colors.

PNG

  • PNG is a versatile image format that supports lossless compression, preserving image quality.
  • It supports transparency.

WEBP

  • WEBP is a modern image format developed by Google that offers both lossy and lossless compression.
  • It provides smaller file sizes compared to JPG and PNG without compromising image quality.
  • WEBP supports transparency and animation.

SVG

  • SVG is a vector image format that uses XML-based code to define shapes, lines, and colors.
  • It scales perfectly to any size without losing quality, making it ideal for responsive design.
  • SVG images are lightweight and can be manipulated with CSS and JavaScript, allowing for interactive and animated graphics on websites.
